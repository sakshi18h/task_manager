# Task Manager (MERN Stack)

## Features
- User Authentication (JWT)
- Role-based access (Admin, Manager, Employee)
- Create, Update, Delete Tasks
- Assign tasks to users

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB

## How to Run

### Backend
cd backend
npm install
npm start

### Frontend
cd frontend
npm install
npm start
